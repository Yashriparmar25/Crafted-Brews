# Crafted Brews 🍰🥖

Crafted Brews is a fully interactive, full-stack e-commerce web application designed for a premium bakery and confectionery experience. Built with a sleek, minimalist aesthetic, it delivers a smooth user journey from product discovery to secure order completion.

## Key Features

* **Dynamic Product Catalog:** Explore beautifully categorized sections including Cakes, Cheesecakes, Artisanal Sourdough, and New Arrivals.
* **Seamless Cart System:** Live cart management allowing users to easily add, modify, and review items in real-time.
* **Comprehensive Checkout Workflow:** A structured, multi-step checkout process covering delivery details validation, address selection, and simulated payment gateways (Credit Card and UPI/QR Code).
* **User Profile & Accounts:** Personalized user dashboards with editable profiles, custom profile avatars, order history tracking, and address book management.
* **Loyalty & Rewards Program:** Integrated customer loyalty tracking system that monitors purchase points and unlocks exclusive rewards.
* **Robust Search Functionality:** Real-time search feature enabling instant product filtering and helpful "no results" handling.
* **Customer Testimonials:** Interactive review slider to display validated community feedback.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Responsive Styling), JavaScript (DOM Manipulation & State Management)
* **Design Aesthetic:** Elegant editorial, high-quality typography, and clean, modern layouts.
